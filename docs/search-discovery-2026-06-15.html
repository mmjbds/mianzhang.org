Google / Bing

Account-console actions.

Google Search Console evidence is accepted only when a dated sitemap receipt, URL-inspection result, or public result URL exists.

Bing Webmaster: submit https://mianzhang.org/sitemap.xml; if a Bing verification file is required, add only the real file provided by Bing. Do not fabricate verification codes.

IndexNow

Usable public endpoint.

IndexNow evidence consists of the live key file, host-specific URL list, and dated API response for that host.

Main-site URL list: https://mianzhang.org/docs/indexnow-url-list-20260615.txt. HF mirror URL list: https://mmjbds-mianzhang-org.static.hf.space/docs/indexnow-url-list-hf-20260615.txt. Submit one host per IndexNow request.
