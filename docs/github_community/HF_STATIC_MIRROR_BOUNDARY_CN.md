# HF 静态镜像边界

Hugging Face 静态镜像只作为公开技术证据的分层索引。它可以链接到 GitHub、Zenodo、自有站和中文区入口，但不应承载官网叙事、中文长文集合、投资人叙事、销售材料或私有沟通材料。

## 推荐分层

- L0 入口层：项目首页、start-here、公开 sitemap、robots、llms。
- L1 证据层：papers、evidence、boundaries、registries、claim-to-evidence。
- L2 复核层：demos、counterexamples、公开复现说明、公开安全样例。
- L3 外部来源层：GitHub 源码与 issue、Zenodo DOI 与归档、自有站入口、中文区入口。

## 不承载

- 官网长文、品牌叙事、投资人叙事、销售材料。
- 中文长文全文搬运；HF 只保留中文区入口。
- 私有 briefing、pilot packet、客户材料、真实交易、内部 agent 编排或商业调度器。
- 论文投稿包、Zenodo 上传包、会议 supplement 包或数据集整包。

## 链接策略

- GitHub 用于源码、issue、反例和贡献流程。
- Zenodo 用于 DOI、公开归档和可引用材料。
- 自有站用于完整公开网站入口。
- 中文区用于中文读者入口，不在 HF README 中展开长文目录。
