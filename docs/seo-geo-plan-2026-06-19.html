SEO / GEO Plan

Scalable search-intent pages with evidence, not empty keyword pages.

The main site uses external hot terms and reader questions as entry points, then routes them into evidence, boundaries, and public challenges. GitHub and Hugging Face remain technical/source/evidence surfaces.

Quality Policy

What this optimization does and does not do.

Scale is necessary for public distribution. Thin duplication is not. Each page answers a real reader question, links to evidence, states boundaries, and offers a counterexample route.

The site does not create low-value doorway abuse, scaled-content abuse, hidden crawler text, fake FAQ, fake schema, or LLM prompt-injection instructions.
