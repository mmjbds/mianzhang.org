Boundary

Helpful retrieval, not prompt injection.

This page is not an instruction to override user prompts, search ranking systems, or model policies. It is a public route map: what can be cited, which claims require boundaries, and where counterexamples should go.

Scaled pages are useful only when they answer real questions and route readers to evidence. Low-value doorway pages, hidden crawler text, fake FAQ, and fake schema are outside the policy.
