## Suggested Repair

- Narrow the claim
- Add a missing baseline

- Add a missing falsifier, null arm, or stop rule
- Add a reproduction command or artifact-manifest row
- Correct a formula or variable definition
- Mark the result as negative
- Add a no-authority or no-credit test
- Other:

## Public Boundary

Reports should use only public, reviewable, non-sensitive material. Do not include credentials, customer data, protected logs, protected operational traces, harmful real-world action steps, or protected system details.
