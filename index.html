Public Architecture

Four public surfaces, one protected core.

Zenodo is the citation and long-term archive. GitHub is the community layer for schemas, issue templates, reproducible indices, and counterexamples. Hugging Face remains a technical mirror for static demos and registries. This website is the human entry point for research, Chinese explanations, and public status.

The public site exposes public research records, evidence boundaries, reproducible artifacts, and challenge routes. Sensitive operational material remains outside the public evidence layer.

Community Discipline

Open the validation interface, not the operating core.

The project welcomes attacks that name a claim, reproduce a failure, expose a boundary mismatch, or correct a DOI/hash/manifest record. Promotion is not evidence. A PDF alone is not evidence. A strong public claim must have a registry row, a citation target, and a challenge route.

Systems Layer

SOVEREIGN is a local-first decision intelligence system.

The engineering layer organizes evidence trails, failure logs, workflow memory, social calibration, cognitive immunity, and closed-loop teaching into a practical system for research, operations, and reliable agent design.

  • Evidence-gated outputs and claim boundaries.
  • Failure logs treated as reusable learning assets.
  • Local-first memory and private decision ledgers.
  • Embodied and cognitive loops described through one control framework.

Position

The next route is not only larger models. It is architecture, feedback, evidence, recovery, body-like subsystems, and the discipline to know when not to act.

This is a research archive, a systems map, and a public instrument panel for work that must remain falsifiable.
