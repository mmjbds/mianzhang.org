Definition

The gate is a record, not a slogan.

An evidence-gated AI system requires high-risk action to pass through public or internal proof fields: claim, support, authority, scope, boundary, receipt, and refusal route.

The public site exposes the research-level evidence and boundaries. It does not expose protected private execution details.

Use

The concept is useful only if it can reject action.

If a gate never stops, narrows, or escalates anything, it is decoration.

A real evidence gate can produce no-action, downgraded language, pending status, or a counterexample task.

FAQ

What is the simplest definition?

Evidence-gated AI requires proof, authority, and scope before high-risk action.

FAQ

What should happen when proof is missing?

The system should stop, downgrade, ask, log, or escalate.
