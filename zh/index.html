核心定位

中文说明与英文证据入口共享同一条公开链路。

mianzhang.org/zh 是中文解释、周报和现实问题入口。它和英文站共享同一套 DOI、sitemap 与 claim boundary，方便读者从中文说明回到可复核的公开记录。

外部分发平台用于触达读者；权威引用仍回到 Zenodo，主张挑战回到 GitHub issue，技术材料回到 Hugging Face 镜像，中文路线和长期说明回到本页。

开放边界

开放可验证接口，同时明确公开边界。

公开层包括概念、论文、公开 schema、演示样例、反例提交模板、证据阶梯和公开 DOI。 未公开系统、敏感数据、运行细节和非公共材料不进入公开发布范围。

  • 公开层用于验证研究主张、复现路径和证据边界，不用于发布完整运行系统。
  • 反例通道把有效质疑转化为可追踪的修正任务。
  • GitHub 负责社区协作，HF 负责技术镜像，中文区负责解释。
  • 公开层只承载可复核材料、证据边界和读者入口。
