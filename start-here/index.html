Start Here

A public evidence field for AI that must earn the right to act.

The project studies learning from failure, proof-carrying action, and relational observability. The public site is not a pitch deck. It is an entry point for claims, artifacts, boundaries, and counterexamples.

Inspect Evidence Attack a Claim Portfolio Archive Index

Position

Simple public rule.

An AI system should not only answer. It should know when it can act, when it must abstain, what evidence supports the action, what would falsify it, what receipt proves what happened, and how to learn without contaminating future decisions. 

claim -> protocol -> evidence -> boundary -> counterexample -> repair

Rule

Boundary first

If a claim requires private material, the public claim is narrowed instead of exposing protected systems.

Operating Sentence

We do not ask people to believe us first; we provide verification, attack, and reproduction entry points first.

That is the public posture: strong claims, narrow boundaries, visible evidence, and a route for correction.
